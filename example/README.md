# Example Usage

This directory contains an example project showing how to use smart-env-validator.

## Setup

1. Copy `.env.example` to `.env.local`
2. Fill in your values
3. Run `npm install` (in the parent directory)
4. Run `npm run build` (in the parent directory)
5. Try the example: `node example/index.js`

## Files

- `env.config.ts` - Schema definition
- `.env.example` - Template for environment variables
- `index.ts` - Example usage in code
